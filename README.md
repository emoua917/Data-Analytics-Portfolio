# Chicago City Budget Dashboard

## Project Overview
An interactive data analytics project analyzing the City of Chicago's 
budget allocations across departments and divisions using Excel and Power BI.

## Data Source
- Source: Chicago Data Portal (data.gov)
- Dataset: 2026 Budget Recommendations
- Size: 8,235 rows of salary and budget data

## Tools Used
- Microsoft Excel Online — Data cleaning, Pivot Tables, Dashboard
- Power BI Desktop — Interactive Dashboard and Visualizations

## Key Insights
- Patrol Services is the highest funded division at $1.22 billion (31% of total budget)
- Chicago Fire Department is second at $556 million (14% of total budget)
- Total city budget across top 10 divisions is $2.74 billion
- Average employee pay rate across all divisions is $100,052
- Highest individual salary recorded is $238,969

## Excel Dashboard Features
- KPI Summary Cards (Total Budget, Total Positions, Avg Pay Rate, Highest Salary)
- Top 10 Divisions by Budget table with Cost Per Position and % of Budget
- Pivot Table with interactive Slicer
- Bar chart and Pie chart visualizations

## Power BI Dashboard Features
- Interactive KPI Cards
- Top 10 Divisions by Budget bar chart
- Budget Share pie chart
- Division budget summary table
- Interactive slicer for filtering by division
